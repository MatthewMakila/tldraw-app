# 📝 Play with TLDraw journal prototype!

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/en/) (version 18+)
- npm or yarn

### Installation
```batch
git clone https://github.com/MatthewMakila/tldraw-app.git
cd tldraw-prototype
npm install
```

## 🔨 Run the Code

### Locally run app in browser
```batch
npm run dev
```
